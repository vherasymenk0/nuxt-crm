### Stack
- Nuxt
- Vue Query
- VeeValidate
- Pinia
- Shadcn
- AppWriteTS

## Development Server

Start the development server on `http://localhost:3000`:

```bash
yarn dev
```
Create shadcn component
```bash
yarn shad:a <name>
```

## Production

```bash
yarn build
```

Locally preview production build:

```bash
yarn preview
```
